# 第三百十一条 （動産の先取特権）

## 
次に掲げる原因によって生じた債権を有する者は、債務者の特定の動産について先取特権を有する。不動産の賃貸借旅館の宿泊旅客又は荷物の運輸動産の保存動産の売買種苗又は肥料（蚕種又は蚕の飼養に供した桑葉を含む。以下同じ。）の供給農業の労務工業の労務

- 一 不動産の賃貸借
- 二 旅館の宿泊
- 三 旅客又は荷物の運輸
- 四 動産の保存
- 五 動産の売買
- 六 種苗又は肥料（蚕種又は蚕の飼養に供した桑葉を含む。以下同じ。）の供給
- 七 農業の労務
- 八 工業の労務
