# 第百五十一条 （株式の質入れの効果）

## 
株式会社が次に掲げる行為をした場合には、株式を目的とする質権は、当該行為によって当該株式の株主が受けることのできる金銭等（金銭その他の財産をいう。以下同じ。）について存在する。[[laws/会社法/本文/第167条.md|第百六十七条]]第一項の規定による取得請求権付株式の取得[[laws/会社法/本文/第170条.md|第百七十条]]第一項の規定による取得条項付株式の取得[[laws/会社法/本文/第173条.md|第百七十三条]]第一項の規定による[[laws/会社法/本文/第171条.md|第百七十一条]]第一項に規定する全部取得条項付種類株式の取得株式の併合株式の分割[[laws/会社法/本文/第185条.md|第百八十五条]]に規定する株式無償割当て[[laws/会社法/本文/第277条.md|第二百七十七条]]に規定する新株予約権無償割当て剰余金の配当残余財産の分配組織変更合併（合併により当該株式会社が消滅する場合に限る。）株式交換株式移転株式の取得（第一号から第三号までに掲げる行為を除く。）

- 一 [[laws/会社法/本文/第167条.md|第百六十七条]]第一項の規定による取得請求権付株式の取得
- 二 [[laws/会社法/本文/第170条.md|第百七十条]]第一項の規定による取得条項付株式の取得
- 三 [[laws/会社法/本文/第173条.md|第百七十三条]]第一項の規定による[[laws/会社法/本文/第171条.md|第百七十一条]]第一項に規定する全部取得条項付種類株式の取得
- 四 株式の併合
- 五 株式の分割
- 六 [[laws/会社法/本文/第185条.md|第百八十五条]]に規定する株式無償割当て
- 七 [[laws/会社法/本文/第277条.md|第二百七十七条]]に規定する新株予約権無償割当て
- 八 剰余金の配当
- 九 残余財産の分配
- 十 組織変更
- 十一 合併（合併により当該株式会社が消滅する場合に限る。）
- 十二 株式交換
- 十三 株式移転
- 十四 株式の取得（第一号から第三号までに掲げる行為を除く。）
## ２
特別支配株主（[[laws/会社法/本文/第179条.md|第百七十九条]]第一項に規定する特別支配株主をいう。[[laws/会社法/本文/第154条.md|第百五十四条]]第三項において同じ。）が株式売渡請求（[[laws/会社法/本文/第179条.md|第百七十九条]]第二項に規定する株式売渡請求をいう。）により売渡株式（[[laws/会社法/本文/第179条の2.md|第百七十九条の二]]第一項第二号に規定する売渡株式をいう。以下この項において同じ。）の取得をした場合には、売渡株式を目的とする質権は、当該取得によって当該売渡株式の株主が受けることのできる金銭について存在する。
