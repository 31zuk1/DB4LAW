# 第四百六十一条 （配当等の制限）

## 
次に掲げる行為により株主に対して交付する金銭等（当該株式会社の株式を除く。以下この節において同じ。）の帳簿価額の総額は、当該行為がその効力を生ずる日における分配可能額を超えてはならない。[[laws/会社法/本文/第138条.md|第百三十八条]]第一号ハ又は第二号ハの請求に応じて行う当該株式会社の株式の買取り[[laws/会社法/本文/第156条.md|第百五十六条]]第一項の規定による決定に基づく当該株式会社の株式の取得（[[laws/会社法/本文/第163条.md|第百六十三条]]に規定する場合又は[[laws/会社法/本文/第165条.md|第百六十五条]]第一項に規定する場合における当該株式会社による株式の取得に限る。）[[laws/会社法/本文/第157条.md|第百五十七条]]第一項の規定による決定に基づく当該株式会社の株式の取得[[laws/会社法/本文/第173条.md|第百七十三条]]第一項の規定による当該株式会社の株式の取得[[laws/会社法/本文/第176条.md|第百七十六条]]第一項の規定による請求に基づく当該株式会社の株式の買取り[[laws/会社法/本文/第197条.md|第百九十七条]]第三項の規定による当該株式会社の株式の買取り[[laws/会社法/本文/第234条.md|第二百三十四条]]第四項（[[laws/会社法/本文/第235条.md|第二百三十五条]]第二項において準用する場合を含む。）の規定による当該株式会社の株式の買取り剰余金の配当

- 一 [[laws/会社法/本文/第138条.md|第百三十八条]]第一号ハ又は第二号ハの請求に応じて行う当該株式会社の株式の買取り
- 二 [[laws/会社法/本文/第156条.md|第百五十六条]]第一項の規定による決定に基づく当該株式会社の株式の取得（[[laws/会社法/本文/第163条.md|第百六十三条]]に規定する場合又は[[laws/会社法/本文/第165条.md|第百六十五条]]第一項に規定する場合における当該株式会社による株式の取得に限る。）
- 三 [[laws/会社法/本文/第157条.md|第百五十七条]]第一項の規定による決定に基づく当該株式会社の株式の取得
- 四 [[laws/会社法/本文/第173条.md|第百七十三条]]第一項の規定による当該株式会社の株式の取得
- 五 [[laws/会社法/本文/第176条.md|第百七十六条]]第一項の規定による請求に基づく当該株式会社の株式の買取り
- 六 [[laws/会社法/本文/第197条.md|第百九十七条]]第三項の規定による当該株式会社の株式の買取り
- 七 [[laws/会社法/本文/第234条.md|第二百三十四条]]第四項（[[laws/会社法/本文/第235条.md|第二百三十五条]]第二項において準用する場合を含む。）の規定による当該株式会社の株式の買取り
- 八 剰余金の配当
## ２
前項に規定する「分配可能額」とは、第一号及び第二号に掲げる額の合計額から第三号から第六号までに掲げる額の合計額を減じて得た額をいう（以下この節において同じ。）。剰余金の額臨時計算書類につき[[laws/会社法/本文/第441条.md|第四百四十一条]]第四項の承認（同項ただし書に規定する場合にあっては、同条第三項の承認）を受けた場合における次に掲げる額[[laws/会社法/本文/第441条.md|第四百四十一条]]第一項第二号の期間の利益の額として法務省令で定める各勘定科目に計上した額の合計額[[laws/会社法/本文/第441条.md|第四百四十一条]]第一項第二号の期間内に自己株式を処分した場合における当該自己株式の対価の額自己株式の帳簿価額最終事業年度の末日後に自己株式を処分した場合における当該自己株式の対価の額第二号に規定する場合における[[laws/会社法/本文/第441条.md|第四百四十一条]]第一項第二号の期間の損失の額として法務省令で定める各勘定科目に計上した額の合計額前三号に掲げるもののほか、法務省令で定める各勘定科目に計上した額の合計額

- 一 剰余金の額
- 二 臨時計算書類につき[[laws/会社法/本文/第441条.md|第四百四十一条]]第四項の承認（同項ただし書に規定する場合にあっては、同条第三項の承認）を受けた場合における次に掲げる額[[laws/会社法/本文/第441条.md|第四百四十一条]]第一項第二号の期間の利益の額として法務省令で定める各勘定科目に計上した額の合計額[[laws/会社法/本文/第441条.md|第四百四十一条]]第一項第二号の期間内に自己株式を処分した場合における当該自己株式の対価の額
- 三 自己株式の帳簿価額
- 四 最終事業年度の末日後に自己株式を処分した場合における当該自己株式の対価の額
- 五 第二号に規定する場合における[[laws/会社法/本文/第441条.md|第四百四十一条]]第一項第二号の期間の損失の額として法務省令で定める各勘定科目に計上した額の合計額
- 六 前三号に掲げるもののほか、法務省令で定める各勘定科目に計上した額の合計額
