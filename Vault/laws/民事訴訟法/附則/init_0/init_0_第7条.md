# 第七条 （送達に関する経過措置）

## 
新法の施行前に裁判所書記官が書類の送達のために郵便を差し出し、又は執行官にその送達の事務を取り扱わせることとした場合には、当該送達については、なお従前の例による。

## ２
新法[[laws/民事訴訟法/本文/第104条.md|第百四条]]第三項の規定は、新法の施行後最初にする送達については、適用しない。

## ３
新法の施行前にした申立てに係る公示送達については、新法[[laws/民事訴訟法/本文/第110条.md|第百十条]]第一項の規定にかかわらず、なお従前の例による。

## ４
新法[[laws/民事訴訟法/本文/第113条.md|第百十三条]]の規定は、新法の施行前に掲示を始めた公示送達については、適用しない。
