# 第八十一条 （他の法令による担保への準用）

## 
[[laws/民事訴訟法/本文/第75条.md|第七十五条]]第四項、第五項及び第七項並びに[[laws/民事訴訟法/本文/第76条.md|第七十六条]]から前条までの規定は、他の法令により訴えの提起について立てるべき担保について準用する。
