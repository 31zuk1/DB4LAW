# 第三条の二 （国民以外の者の国外犯）

## 
この法律は、日本国外において日本国民に対して次に掲げる罪を犯した日本国民以外の者に適用する。[[laws/刑法/本文/第176条.md|第百七十六条]]、[[laws/刑法/本文/第177条.md|第百七十七条]]及び[[laws/刑法/本文/第179条.md|第百七十九条]]から[[laws/刑法/本文/第181条.md|第百八十一条]]まで（不同意わいせつ、不同意性交等、監護者わいせつ及び監護者性交等、未遂罪、不同意わいせつ等致死傷）の罪[[laws/刑法/本文/第199条.md|第百九十九条]]（殺人）の罪及びその未遂罪[[laws/刑法/本文/第204条.md|第二百四条]]（傷害）及び[[laws/刑法/本文/第205条.md|第二百五条]]（傷害致死）の罪[[laws/刑法/本文/第220条.md|第二百二十条]]（逮捕及び監禁）及び[[laws/刑法/本文/第221条.md|第二百二十一条]]（逮捕等致死傷）の罪[[laws/刑法/本文/第224条.md|第二百二十四条]]から[[laws/刑法/本文/第228条.md|第二百二十八条]]まで（未成年者略取及び誘拐、営利目的等略取及び誘拐、身の代金目的略取等、所在国外移送目的略取及び誘拐、人身売買、被略取者等所在国外移送、被略取者引渡し等、未遂罪）の罪[[laws/刑法/本文/第236条.md|第二百三十六条]]（強盗）、[[laws/刑法/本文/第238条.md|第二百三十八条]]から[[laws/刑法/本文/第240条.md|第二百四十条]]まで（事後強盗、昏酔強盗、強盗致死傷）並びに[[laws/刑法/本文/第241条.md|第二百四十一条]]第一項及び第三項（強盗・不同意性交等及び同致死）の罪並びにこれらの罪（同条第一項の罪を除く。）の未遂罪

- 一 [[laws/刑法/本文/第176条.md|第百七十六条]]、[[laws/刑法/本文/第177条.md|第百七十七条]]及び[[laws/刑法/本文/第179条.md|第百七十九条]]から[[laws/刑法/本文/第181条.md|第百八十一条]]まで（不同意わいせつ、不同意性交等、監護者わいせつ及び監護者性交等、未遂罪、不同意わいせつ等致死傷）の罪
- 二 [[laws/刑法/本文/第199条.md|第百九十九条]]（殺人）の罪及びその未遂罪
- 三 [[laws/刑法/本文/第204条.md|第二百四条]]（傷害）及び[[laws/刑法/本文/第205条.md|第二百五条]]（傷害致死）の罪
- 四 [[laws/刑法/本文/第220条.md|第二百二十条]]（逮捕及び監禁）及び[[laws/刑法/本文/第221条.md|第二百二十一条]]（逮捕等致死傷）の罪
- 五 [[laws/刑法/本文/第224条.md|第二百二十四条]]から[[laws/刑法/本文/第228条.md|第二百二十八条]]まで（未成年者略取及び誘拐、営利目的等略取及び誘拐、身の代金目的略取等、所在国外移送目的略取及び誘拐、人身売買、被略取者等所在国外移送、被略取者引渡し等、未遂罪）の罪
- 六 [[laws/刑法/本文/第236条.md|第二百三十六条]]（強盗）、[[laws/刑法/本文/第238条.md|第二百三十八条]]から[[laws/刑法/本文/第240条.md|第二百四十条]]まで（事後強盗、昏酔強盗、強盗致死傷）並びに[[laws/刑法/本文/第241条.md|第二百四十一条]]第一項及び第三項（強盗・不同意性交等及び同致死）の罪並びにこれらの罪（同条第一項の罪を除く。）の未遂罪
