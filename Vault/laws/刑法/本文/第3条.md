# 第三条 （国民の国外犯）

## 
この法律は、日本国外において次に掲げる罪を犯した日本国民に適用する。[[laws/刑法/本文/第108条.md|第百八条]]（現住建造物等放火）及び[[laws/刑法/本文/第109条.md|第百九条]]第一項（非現住建造物等放火）の罪、これらの規定の例により処断すべき罪並びにこれらの罪の未遂罪[[laws/刑法/本文/第119条.md|第百十九条]]（現住建造物等浸害）の罪[[laws/刑法/本文/第159条.md|第百五十九条]]から[[laws/刑法/本文/第161条.md|第百六十一条]]まで（私文書偽造等、虚偽診断書等作成、偽造私文書等行使）及び前条第五号に規定する電磁的記録以外の電磁的記録に係る[[laws/刑法/本文/第161条.md|第百六十一条]]の二の罪[[laws/刑法/本文/第167条.md|第百六十七条]]（私印偽造及び不正使用等）の罪及び同条第二項の罪の未遂罪[[laws/刑法/本文/第176条.md|第百七十六条]]、[[laws/刑法/本文/第177条.md|第百七十七条]]及び[[laws/刑法/本文/第179条.md|第百七十九条]]から[[laws/刑法/本文/第181条.md|第百八十一条]]まで（不同意わいせつ、不同意性交等、監護者わいせつ及び監護者性交等、未遂罪、不同意わいせつ等致死傷）並びに[[laws/刑法/本文/第184条.md|第百八十四条]]（重婚）の罪[[laws/刑法/本文/第198条.md|第百九十八条]]（贈賄）の罪[[laws/刑法/本文/第199条.md|第百九十九条]]（殺人）の罪及びその未遂罪[[laws/刑法/本文/第204条.md|第二百四条]]（傷害）及び[[laws/刑法/本文/第205条.md|第二百五条]]（傷害致死）の罪[[laws/刑法/本文/第214条.md|第二百十四条]]から[[laws/刑法/本文/第216条.md|第二百十六条]]まで（業務上堕胎及び同致死傷、不同意堕胎、不同意堕胎致死傷）の罪[[laws/刑法/本文/第218条.md|第二百十八条]]（保護責任者遺棄等）の罪及び同条の罪に係る[[laws/刑法/本文/第219条.md|第二百十九条]]（遺棄等致死傷）の罪[[laws/刑法/本文/第220条.md|第二百二十条]]（逮捕及び監禁）及び[[laws/刑法/本文/第221条.md|第二百二十一条]]（逮捕等致死傷）の罪[[laws/刑法/本文/第224条.md|第二百二十四条]]から[[laws/刑法/本文/第228条.md|第二百二十八条]]まで（未成年者略取及び誘拐、営利目的等略取及び誘拐、身の代金目的略取等、所在国外移送目的略取及び誘拐、人身売買、被略取者等所在国外移送、被略取者引渡し等、未遂罪）の罪[[laws/刑法/本文/第230条.md|第二百三十条]]（名誉毀損）の罪[[laws/刑法/本文/第235条.md|第二百三十五条]]から[[laws/刑法/本文/第236条.md|第二百三十六条]]まで（窃盗、不動産侵奪、強盗）、[[laws/刑法/本文/第238条.md|第二百三十八条]]から[[laws/刑法/本文/第240条.md|第二百四十条]]まで（事後強盗、昏こん酔強盗、強盗致死傷）、[[laws/刑法/本文/第241条.md|第二百四十一条]]第一項及び第三項（強盗・不同意性交等及び同致死）並びに[[laws/刑法/本文/第243条.md|第二百四十三条]]（未遂罪）の罪[[laws/刑法/本文/第246条.md|第二百四十六条]]から[[laws/刑法/本文/第250条.md|第二百五十条]]まで（詐欺、電子計算機使用詐欺、背任、準詐欺、恐喝、未遂罪）の罪[[laws/刑法/本文/第253条.md|第二百五十三条]]（業務上横領）の罪[[laws/刑法/本文/第256条.md|第二百五十六条]]第二項（盗品譲受け等）の罪

- 一 [[laws/刑法/本文/第108条.md|第百八条]]（現住建造物等放火）及び[[laws/刑法/本文/第109条.md|第百九条]]第一項（非現住建造物等放火）の罪、これらの規定の例により処断すべき罪並びにこれらの罪の未遂罪
- 二 [[laws/刑法/本文/第119条.md|第百十九条]]（現住建造物等浸害）の罪
- 三 [[laws/刑法/本文/第159条.md|第百五十九条]]から[[laws/刑法/本文/第161条.md|第百六十一条]]まで（私文書偽造等、虚偽診断書等作成、偽造私文書等行使）及び前条第五号に規定する電磁的記録以外の電磁的記録に係る[[laws/刑法/本文/第161条.md|第百六十一条]]の二の罪
- 四 [[laws/刑法/本文/第167条.md|第百六十七条]]（私印偽造及び不正使用等）の罪及び同条第二項の罪の未遂罪
- 五 [[laws/刑法/本文/第176条.md|第百七十六条]]、[[laws/刑法/本文/第177条.md|第百七十七条]]及び[[laws/刑法/本文/第179条.md|第百七十九条]]から[[laws/刑法/本文/第181条.md|第百八十一条]]まで（不同意わいせつ、不同意性交等、監護者わいせつ及び監護者性交等、未遂罪、不同意わいせつ等致死傷）並びに[[laws/刑法/本文/第184条.md|第百八十四条]]（重婚）の罪
- 六 [[laws/刑法/本文/第198条.md|第百九十八条]]（贈賄）の罪
- 七 [[laws/刑法/本文/第199条.md|第百九十九条]]（殺人）の罪及びその未遂罪
- 八 [[laws/刑法/本文/第204条.md|第二百四条]]（傷害）及び[[laws/刑法/本文/第205条.md|第二百五条]]（傷害致死）の罪
- 九 [[laws/刑法/本文/第214条.md|第二百十四条]]から[[laws/刑法/本文/第216条.md|第二百十六条]]まで（業務上堕胎及び同致死傷、不同意堕胎、不同意堕胎致死傷）の罪
- 十 [[laws/刑法/本文/第218条.md|第二百十八条]]（保護責任者遺棄等）の罪及び同条の罪に係る[[laws/刑法/本文/第219条.md|第二百十九条]]（遺棄等致死傷）の罪
- 十一 [[laws/刑法/本文/第220条.md|第二百二十条]]（逮捕及び監禁）及び[[laws/刑法/本文/第221条.md|第二百二十一条]]（逮捕等致死傷）の罪
- 十二 [[laws/刑法/本文/第224条.md|第二百二十四条]]から[[laws/刑法/本文/第228条.md|第二百二十八条]]まで（未成年者略取及び誘拐、営利目的等略取及び誘拐、身の代金目的略取等、所在国外移送目的略取及び誘拐、人身売買、被略取者等所在国外移送、被略取者引渡し等、未遂罪）の罪
- 十三 [[laws/刑法/本文/第230条.md|第二百三十条]]（名誉毀損）の罪
- 十四 [[laws/刑法/本文/第235条.md|第二百三十五条]]から[[laws/刑法/本文/第236条.md|第二百三十六条]]まで（窃盗、不動産侵奪、強盗）、[[laws/刑法/本文/第238条.md|第二百三十八条]]から[[laws/刑法/本文/第240条.md|第二百四十条]]まで（事後強盗、昏こん酔強盗、強盗致死傷）、[[laws/刑法/本文/第241条.md|第二百四十一条]]第一項及び第三項（強盗・不同意性交等及び同致死）並びに[[laws/刑法/本文/第243条.md|第二百四十三条]]（未遂罪）の罪
- 十五 [[laws/刑法/本文/第246条.md|第二百四十六条]]から[[laws/刑法/本文/第250条.md|第二百五十条]]まで（詐欺、電子計算機使用詐欺、背任、準詐欺、恐喝、未遂罪）の罪
- 十六 [[laws/刑法/本文/第253条.md|第二百五十三条]]（業務上横領）の罪
